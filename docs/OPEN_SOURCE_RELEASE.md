# Open-Source Release Notes

This tree is a sanitized publication workspace generated from the active
ForensiFlow experiment workspace.

Excluded from the public tree:

- API keys and local assistant state: `.env`, `.env.*`, `.codex*`, `.profile`,
  `memory/`, `.claude/`.
- Private evidence and experiment data: `data/`, `artifacts/`, `experiments/`,
  XML dumps, screenshots, and run logs.
- Large or local-only assets: model weights, frontend build outputs,
  `node_modules/`, backup directories, and nested third-party repositories.
- Historical generated scripts and private RAG registry entries.

The active experiment workspace remains separate and can continue to be used for
device testing. Push only this sanitized workspace when publishing.
